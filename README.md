# Test Questions

This repository contains my personal responses to interview/sharing questions about how I use AI in my daily work.

## Topics Covered

1. **What tasks have I completed using AI?**
   - Automated due diligence with a 4-agent system on Paperclip
   - Trend research & product idea generation
   - From idea to MVP using Claude CLI

2. **My workflow when using AI**
   - Building a personal knowledge base as trusted source
   - Choosing the right model based on task complexity
   - Using specialized skill templates per task type

3. **How I handle problems**
   - Using Grok for market research & real-time data
   - Using Claude for bug fixing & vibe coding
   - Always validating AI output before handoff

4. **What is the optimal approach?**
   - Knowledge base as foundation
   - Right model for the right task
   - Specialized skills over generic prompts
